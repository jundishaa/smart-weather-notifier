About This Project

The Smart Weather Notifier was inspired by the need for accurate, real-time weather updates, especially for people living in areas with unpredictable weather. Built as part of my portfolio for the ALX Software Engineering program, this project combines Flask, MySQL, and React to provide users with timely weather notifications.
